Equation of plane is : Ax + By + C = 0   (in 2 dim)   or Wx+b = 0 in N dim  (W is a vector)
Unit vector normal to plane is W/||w||, ie [A B]/sqrt(A^2+B^2)
Distance from x0, y0 to plane is: (Ax0 + By0 + C)/sqrt(A^2+B^2)
Distance from origin to plane is: C/sqrt(A^2+B^2)
